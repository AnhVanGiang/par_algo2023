## Getting started

This Repository contains instructions and examples on

- how to work on the Snellius supercomputer
- how to use MPI and/or BSP there
- where to find more information on the practical part of the lectures

The repository is updated as we go along. To clone it initially on Snellius, use
```bash
$ git clone https://github.com/jthies/par_algo2023
```
To update your local repository, go to the cloned directory and type ``git pull``.
